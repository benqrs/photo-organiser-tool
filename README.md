# Photo Organiser Tool

Organise and rename portrait session photos by subject — using EXIF copyright data or filename patterns.

**Download the latest version from the [Releases](../../releases) page.**

### Install Instructions
1. Open the DMG and drag **Photo Organiser Tool.app** into your Applications folder
2. Open Terminal and run:
   ```
   xattr -cr "/Applications/Photo Organiser Tool.app"
   ```
3. Open Photo Organiser Tool from your Applications folder

Requires macOS with Apple Silicon (M1 or later).
